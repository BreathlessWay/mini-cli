### 小程序 CLI

[从零开始做一个小程序脚手架](http://breathlessway.com/read/detail/60791ec14aebf60064f670ed)

> 使用

1. 安装 `npm i @breathlessway/mini-cli -g`
2. 命令
    - `mini -h`: 帮助
    - `mini -v`: 版本
    - `mini -g <path>`: 生成 babel helper 和 regeneratorRuntime 到指定目录
    - `mini -i`: 创建项目
